# kallin.github.io

My personal site and blog: writing on software engineering, AI coding-agent tooling, and civic tech.

**Live site:** https://kallin.github.io

## Stack

Built with [Astro](https://astro.build). Blog posts live in `src/content/blog/` as Markdown / MDX, and the site deploys to GitHub Pages on push to `master`.

## Local development

```bash
npm install
npm run dev      # dev server at http://localhost:4321
npm run build    # production build to ./dist
```

Node 22+ (managed via `.mise.toml`).
